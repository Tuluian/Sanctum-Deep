# Story 16.5: Steam Store Page Assets

## Status: Draft

## Story

**As a** developer publishing on Steam,
**I want** all store page assets created and optimized,
**so that** the game presents professionally and attracts players.

## Acceptance Criteria

1. Store header image (capsule) in all required sizes
2. Screenshots showing key gameplay (5-10)
3. Store description written and formatted
4. Short description/tagline complete
5. Trailer video created (optional but recommended)
6. Library assets (hero, logo, icons)
7. Community assets (avatars, backgrounds)

## Offline Work Tasks

All design and writing work can be done offline.

### 1. Capsule Images (2-3 hours)

Steam requires multiple capsule sizes:

| Asset | Size | Use |
|-------|------|-----|
| Header Capsule | 460 x 215 | Store page header |
| Small Capsule | 231 x 87 | Lists, search results |
| Main Capsule | 616 x 353 | Featured spots |
| Hero Capsule | 1920 x 620 | Large feature |
| Vertical Capsule | 374 x 448 | Collections |

**Design Requirements:**
- Game title clearly visible
- No "Coming Soon" or "Early Access" text (unless permanent)
- Key art/character visible
- Consistent branding across all sizes
- PNG or JPG format

**Content Suggestions:**
- Main character or class selection
- Game logo prominently displayed
- Dark, mysterious atmosphere matching game
- Tagline: "Roguelike Deck-Builder"

### 2. Screenshots (2-3 hours)

Create 5-10 screenshots showing:

1. **Combat Scene** - Card hand, enemies, HP bars
2. **Card Selection** - Reward screen or deck view
3. **Class Selection** - Multiple classes shown
4. **Map/Progression** - Roguelike map with branching paths
5. **Boss Fight** - Dramatic boss encounter
6. **Card Effects** - Flashy ability or status effect
7. **Upgrade Screen** - Meta-progression system
8. **Victory Screen** - Satisfying completion moment

**Screenshot Requirements:**
- Size: 1920 x 1080 (1080p) minimum
- Format: PNG (no compression artifacts)
- No UI that covers key content
- Capture at game's best visual moments
- Can add subtle promotional overlays

**Capture Tips:**
- Use browser at 1920x1080 or higher
- Use DevTools to hide UI elements if needed
- Take multiple shots, choose best composition
- Consider adding subtle vignette or polish

## Dev Notes

### Asset Templates
Consider creating a Figma/Photoshop template with:
- All capsule sizes as artboards
- Brand colors and fonts
- Logo in vector format
- Reusable background elements

### Image Optimization
- Use PNG for screenshots (lossless)
- Use JPG for capsules if file size is issue
- Max file sizes: 2MB per image
- Color profile: sRGB

### Trailer Video (Optional but Valuable)
If creating a trailer:
- Length: 30-90 seconds
- Resolution: 1920x1080 minimum
- Format: MP4, H.264
- Show gameplay, not just cinematics
- Include audio (game music/SFX)
- End with logo and call-to-action

### Steam Direct Submission
- $100 fee per game (recoupable at ~$1000 revenue)
- Tax forms required (W-8BEN or W-9)
- Bank account for payments
- Review process: ~2-5 business days
