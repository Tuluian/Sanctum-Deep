# Story 7.8: Card Text Centering

## Status: Draft

## Story

**As a** player viewing cards in my hand,
**I want** card ability text and category text to be centered,
**so that** cards look polished and professional (name remains left-aligned).

## Acceptance Criteria

1. Card ability/effect text is horizontally centered
2. Card category/type text is horizontally centered
3. Card name remains left-aligned (current behavior)
4. Centering works for all card text lengths
5. Multi-line ability text centers each line
6. Consistent alignment across all card views (hand, rewards, deck viewer)

  - [ ] Handle multi-line wrapping
- [ ] Update card CSS for category text (AC: 2)
