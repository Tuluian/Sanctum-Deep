# Story 12.1: Visual Asset Acquisition & Emoji Replacement

## Status: Draft

## Story

**As a** player of Sanctum Ruins,
**I want** professional visual assets instead of emoji placeholders,
**so that** the game looks polished and immersive.

## Acceptance Criteria

1. Research and catalog available free assets (Unity Store, itch.io, OpenGameArt)
2. Identify paid assets worth purchasing (budget TBD)
3. Replace enemy emoji placeholders with proper sprites
4. Replace card icons with proper art
5. Replace UI components with styled elements
6. Consistent art style across all assets
7. Assets are properly licensed for commercial use

## Asset Categories Needed

### Enemies (Priority: High)
- Act 1: Skeleton, Cultist, Ghoul, Bone Archer, Shade, Tomb Guardian, High Cultist, Bonelord
- Act 2: Slime, Dark Mage, Gargoyle, Corrupted Spirit, etc.
- Act 3: Imp, Corrupted Angel, Void Spawn, Sanctum Guardian, etc.

### UI Elements (Priority: Medium)
- Card frames (per class color scheme)
- HP/Resolve bars
- Button styles
- Map node icons
- Status effect icons

### Characters (Priority: Medium)
- Class portraits for selection screen
- Class icons for cards

### Environment (Priority: Low)
- Combat backgrounds (Act 1, 2, 3)
- Map backgrounds
