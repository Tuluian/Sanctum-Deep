# Story 15.6: SEO & Social Meta Tags

## Status: InProgress

## Story

**As a** game publisher,
**I want** proper meta tags and SEO configuration,
**so that** the game appears well in search results and social shares.

## Acceptance Criteria

1. Title and description meta tags optimized
2. Open Graph tags for social sharing (Facebook, Discord)
3. Twitter Card tags configured
4. Structured data (JSON-LD) for rich search results
5. Favicon and touch icons complete
6. robots.txt and sitemap.xml present
7. Social share preview looks professional

## Offline Work Tasks

All tasks can be done offline with local testing.

### 1. Update index.html Meta Tags (30 min)

Add/update in `index.html` `<head>`:

```html
<!-- Primary Meta Tags -->
<title>Sanctum Ruins - Roguelike Deck-Builder Game</title>
<meta name="title" content="Sanctum Ruins - Roguelike Deck-Builder Game">
<meta name="description" content="A strategic roguelike deck-builder inspired by Slay the Spire. Build your deck, defeat enemies, and conquer the Sanctum in quick 15-30 minute sessions.">
<meta name="keywords" content="deck builder, roguelike, card game, strategy, fantasy, indie game, slay the spire, browser game">
<meta name="author" content="[Your Name/Studio]">
<meta name="robots" content="index, follow">
<link rel="canonical" href="https://[YOUR_DOMAIN]/">

<!-- Open Graph / Facebook -->
<meta property="og:type" content="website">
<meta property="og:url" content="https://[YOUR_DOMAIN]/">
<meta property="og:title" content="Sanctum Ruins - Roguelike Deck-Builder Game">
<meta property="og:description" content="A strategic roguelike deck-builder inspired by Slay the Spire. Build your deck, defeat enemies, and conquer the Sanctum.">
<meta property="og:image" content="https://[YOUR_DOMAIN]/images/og-image.png">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:site_name" content="Sanctum Ruins">
<meta property="og:locale" content="en_US">

<!-- Twitter -->
<meta property="twitter:card" content="summary_large_image">
<meta property="twitter:url" content="https://[YOUR_DOMAIN]/">
<meta property="twitter:title" content="Sanctum Ruins - Roguelike Deck-Builder Game">
<meta property="twitter:description" content="A strategic roguelike deck-builder inspired by Slay the Spire. Build your deck, defeat enemies, and conquer the Sanctum.">
<meta property="twitter:image" content="https://[YOUR_DOMAIN]/images/twitter-card.png">

<!-- Discord (uses OG tags primarily, but also reads these) -->
<meta name="theme-color" content="#16213e">
```

### 2. Create Social Share Images (1-2 hours)

Create promotional images:

**Open Graph Image** (`public/images/og-image.png`):
- Size: 1200 x 630 pixels
- Content: Game logo, tagline, gameplay screenshot
- File size: Under 1MB for fast loading

**Twitter Card** (`public/images/twitter-card.png`):
- Size: 1200 x 628 pixels (or use same as OG)
- Can be same image as OG

**Design Tips**:
- Use game art/screenshots as background
- Include "Sanctum Ruins" title prominently
- Add tagline: "Roguelike Deck-Builder"
- Dark theme to match game aesthetic
- Text should be readable at small sizes

### 3. Create Structured Data (30 min)

Add JSON-LD in `index.html` before `</head>`:

```html
<!-- Structured Data -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "VideoGame",
  "name": "Sanctum Ruins",
  "description": "A strategic roguelike deck-builder inspired by Slay the Spire with 8 unique character classes.",
  "url": "https://[YOUR_DOMAIN]/",
  "image": "https://[YOUR_DOMAIN]/images/og-image.png",
  "genre": ["Card Game", "Roguelike", "Strategy"],
  "gamePlatform": ["Web Browser", "iOS"],
  "applicationCategory": "Game",
  "operatingSystem": "Any",
  "offers": {
    "@type": "Offer",
    "price": "0",
    "priceCurrency": "USD",
    "availability": "https://schema.org/InStock"
  },
  "author": {
    "@type": "Organization",
    "name": "[Your Studio Name]"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5",
    "reviewCount": "100"
  }
}
</script>
```

**Note**: Remove aggregateRating until you have actual reviews.

### 4. Create robots.txt (10 min)

Create `public/robots.txt`:

```
# Sanctum Ruins robots.txt

User-agent: *
Allow: /
Disallow: /api/
Disallow: /admin/

# Sitemap location
Sitemap: https://[YOUR_DOMAIN]/sitemap.xml
```

### 5. Create sitemap.xml (15 min)

Create `public/sitemap.xml`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://[YOUR_DOMAIN]/</loc>
    <lastmod>2025-12-05</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://[YOUR_DOMAIN]/privacy</loc>
    <lastmod>2025-12-05</lastmod>
    <changefreq>yearly</changefreq>
    <priority>0.3</priority>
  </url>
  <url>
    <loc>https://[YOUR_DOMAIN]/terms</loc>
    <lastmod>2025-12-05</lastmod>
    <changefreq>yearly</changefreq>
    <priority>0.3</priority>
  </url>
</urlset>
```

### 6. Complete Favicon Set (30 min)

Ensure these exist in `public/`:

```
favicon.ico          (16x16, 32x32 multi-size)
favicon-16x16.png
favicon-32x32.png
apple-touch-icon.png (180x180)
safari-pinned-tab.svg (optional, monochrome)
```

Add to `index.html`:

```html
<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
```

**Tool**: Use https://realfavicongenerator.net/ with a 512x512 source image

### 7. Test Social Previews

After deploying, test with:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/

For local testing, check meta tags in browser DevTools.

## Dev Notes

### Image Optimization
- Use PNG for images with text
- Compress images (TinyPNG or similar)
- Keep OG image under 1MB
- Use WebP with PNG fallback if supported

### SEO Best Practices
- Title: 50-60 characters
- Description: 150-160 characters
- Keywords: 5-10 relevant terms
- Canonical URL: Prevent duplicate content

### Social Platform Requirements

| Platform | Image Size | Notes |
|----------|------------|-------|
| Facebook | 1200x630 | og:image |
| Twitter | 1200x628 | Large card |
| LinkedIn | 1200x627 | Uses OG |
| Discord | 1200x630 | Uses OG |

### Placeholder Replacement
Before publishing, replace:
- `[YOUR_DOMAIN]` - Your actual domain
- `[Your Name/Studio]` - Your name or studio
- `[Your Studio Name]` - Studio name for structured data
